About This Project

LinguaHaru is an AI-powered document translation application that makes translating documents effortless. Unlike simple text translators, it understands document structure and preserves formatting while translating content.

The application features a beautiful Gradio-based interface with support for 13 languages, multiple AI providers (ChatGPT, Claude, DeepSeek, Gemini, etc.), and even offline translation using local models via Ollama.
